perform

infix fun perform(callback: () -> Unit): Unit

Perform a blocking action such that it is within a time context. This requires a TimePerformer such that the context of how to time the performance, whether it be an initial delay, exact time execution, and/or some kind of iteration count based on how long to run, how often to run, how to run, when to run.

val start = now
(start until (10.minutes from start)         //from start/[now] until 10.minutes from start/[now]
     every (1.minutes.exactly                //Every 1 minute exactly
         (waitUntil 1.minutes)               //wait until 1 minute of the hour
         (starting (1.minutes from start)))  //starting 1 minute from start/now
     .perform{                               //Perform this...
         println("Hello, world!")
     }
