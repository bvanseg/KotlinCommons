This should be done using a flatMap operator. Add an abstract method flatMap to TimeContainer such that all TimeContainers can be flatMap'd. We also need a way of constructing timeObjects directly into a TimeContainer, perhaps as a primary constructor and have a secondary constructor for default time objects.

//Flatmap so we can transform timeObject property from old (passed into callback) to new (returned from callback)
val new = now.flatMap{
    Time(
         it.year,
         it.month,
         it.day,
         it.hour,
         it.minute,
         it.seconds + context.value,
         it.nano
    )
} //Should produce the same thing as [now] except it should keep the rest of it's numbers
