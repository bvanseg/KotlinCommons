softInvoke

fun softInvoke(args: String, context: Context): Int

Functions similarly to {@link InternalCommand#invoke()}, but does not actually execute the function. Instead, it tracks arguments and returns a score based on the argument chain. This is used to help a {@link CommandModule} determine what command is best fit to handle a set of arguments.
